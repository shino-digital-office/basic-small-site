# starterkit_v3

前バージョンv2からの変更点
・v3はcatalogのbg,cardを無効化した。(上書きが面倒なため)
・見本(example)を削除した(軽量化のため)
・catalogをcommonに名前変更した。カタログという位置づけからscssの分割管理という思想にシフトした。
・scssをパーツ毎に切り分け、どこに何がかいてあるかすぐわかるようにした。
・swiperの初期設定の漏れがないように予め必要なコードを書き込んでおいた。
・要素命名ルールの曖昧さを減らした。（設計意図を命名に込めた）

このリポジトリは「完成品テンプレート」ではない。  
ここは、すべての派生テンプレートと案件構造を生み出すための  
**設計思想の核（core）** である。

この リポジトリ に含まれるのは、  
見た目の完成度ではなく、構造・レイヤー・責務分離・UI エンジン設計など、  
今後すべての制作物の“骨格”になるルールだけである。

したがって、このリポジトリは直接案件には使用しない。  
この リポジトリ を基点として派生リポジトリを作成し、  
用途別テンプレート（blog / lp / shop / portfolio など）を生成する。

この リポジトリ は「作るための道具」であり、  
「使うための完成品」ではない。
